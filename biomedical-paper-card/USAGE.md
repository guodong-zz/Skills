# Biomedical Paper Card — Usage Guide

## Recommended Prompt for a New Chat

请读取我已连接的 GitHub 仓库 `guodong-zz/AI-Skills` 中的：

`biomedical-paper-card/SKILL.md`

严格按照该 Skill 中定义的 Biomedical Paper Card 工作流分析我接下来提供的论文。

要求：

- 所有具体结论优先基于论文原文证据
- 不要仅依赖摘要
- 涉及 Figure 时尽可能对应具体 Figure 编号
- 不得虚构论文中未报告的实验或结果
- 如果信息不足，请明确说明
- 后续追问仍需优先回到原始论文内容，而不是只依赖 Paper Card

现在先读取并加载该 Skill，然后等待我上传论文。

回答时优先引用论文原始证据，并指出对应 Figure 或实验。

## Multi-paper Prompt

请读取并使用：

`guodong-zz/AI-Skills/biomedical-paper-card/SKILL.md`

分别分析我提供的每篇论文，并为每篇论文生成独立的 Paper Card。

在完成单篇分析后，再根据各自 Paper Card 进行横向比较。

比较时重点关注：

- Core scientific question
- Study design
- Key findings
- Mechanistic model
- Limitations
- Novelty
- Relevance to my research
